# SMTP(ShowMeThePage)

## What is this?
* SMPT는 markdown으로 작성하는 Web SlideShow Tool입니다.
* 프리젠테이션 시 청중의 반응을 시각적으로 확인할 수 있습니다.
* 스마트폰으로 프리젠테이션의 만족도를 표현할 수 있습니다.

## 사용된 라이브러리
* Client Javascript Library
 * Twitter BootStrap
 * CodeMirror2
 * reveal.js
 * jquery
 * ball_pool
* Server Library
 * Node
 * Socket.io
 * Express
 * jqtpl
 * mongolian
 * shorturl
 * github-flavored-markdown
